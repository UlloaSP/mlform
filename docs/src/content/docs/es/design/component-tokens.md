---
title: Component tokens
description: Overrides por familia de componente.
---

```ts
designSystem: {
  overrides: {
    components: {
      submit: {
        tokens: {
          "--mlf-submit-radius": "6px",
          "--mlf-submit-shadow": "none"
        }
      }
    }
  }
}
```

Usa tokens globales para marca y tokens de componente para ajustar una pieza concreta.
