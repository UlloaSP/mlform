---
title: Layout Tabs
description: Usa el layout oficial de tabs o el mismo config mediante la API headless.
---

`mountTabsForm()` monta la UI oficial de tabs.

Caracteristicas:

- tablist superior
- solo renderiza el tab activo
- footer persistente con submit
- navegar entre tabs no valida

Usa `createFormView()` si quieres tabs propios con shell de aplicacion.
