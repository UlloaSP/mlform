---
title: Layout Accordion
description: Usa el layout oficial accordion o el mismo config mediante la API headless.
---

`mountAccordionForm()` monta la UI oficial de accordion.

Caracteristicas:

- secciones desplegables de arriba a abajo
- multiples secciones pueden quedar abiertas
- la primera abre por defecto salvo override
- footer persistente con submit
- abrir o cerrar secciones no valida
