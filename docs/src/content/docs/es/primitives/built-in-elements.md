---
title: Elementos integrados
description: Custom elements registrados por MLForm primitives.
---

| Elemento                    | Propósito                             |
| --------------------------- | ------------------------------------- |
| `mlf-form`                  | Host principal.                       |
| `mlf-form-errors`           | Errores de formulario.                |
| `mlf-submit-button`         | Botón de submit.                      |
| `mlf-unsupported-component` | Fallback para renderers desconocidos. |
| `mlf-field-frame`           | Wrapper de campo.                     |
| `mlf-report-frame`          | Wrapper de informe.                   |
| `mlf-text-field`            | Campo text.                           |
| `mlf-number-field`          | Campo number.                         |
| `mlf-boolean-field`         | Campo boolean.                        |
| `mlf-category-field`        | Campo category.                       |
| `mlf-date-field`            | Campo date.                           |
| `mlf-time-series-field`     | Campo time-series.                    |
| `mlf-classifier-report`     | Informe classifier.                   |
| `mlf-regressor-report`      | Informe regressor.                    |
