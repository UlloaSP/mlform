---
title: Layouts
description: Opciones de layout para formulario e informes.
---

| Opción       | Valores                    | Uso                                |
| ------------ | -------------------------- | ---------------------------------- |
| `layout`     | `stacked`, `split`         | Disposición general.               |
| `reportPane` | `auto`, `always`, `hidden` | Visibilidad del panel de informes. |

Usa `stacked` para embeds estrechos y `split` cuando los informes son centrales.
